# 🇯🇵 京都大阪七天六夜之旅 PWA

一個漸進式網頁應用程式（PWA），提供 2026 年 1 月 16 日至 22 日的京都和大阪旅遊行程規劃。

## 📋 功能特色

- ✅ 完整的七天六夜行程規劃
- ✅ 包含交通、住宿、景點、美食資訊
- ✅ PWA 支援 - 可安裝到手機桌面
- ✅ 離線瀏覽功能
- ✅ 響應式設計 - 支援手機、平板、電腦
- ✅ 觸控手勢導航
- ✅ 鍵盤快捷鍵支援

## 💻 技術架構

- **HTML5** - 語義化標記
- **CSS3** - 響應式設計、動畫效果

- **Service Worker** - 離線快取、後台同步

## 🚀 快速開始

### 線上訪問

直接開啟 `index.html` 檔案或部署到任何網頁伺服器。

### 本地開發

```bash
# 使用 Python 簡易伺服器
python -m http.server 8000

# 或使用 Node.js
npx http-server
```

然後在瀏覽器開啟 `http://localhost:8000`

### 安裝為 PWA

1. 在支援的瀏覽器中開啟網站
2. 點擊「安裝」按鈕或瀏覽器的「安裝應用程式」選項
3. 應用程式將新增到您的主畫面

## 📝 旅遊小貼士

- 建議購買關西周遊卡或 JR Pass
- 1 月為京都淡季，遊客較少但需注意保暖
- 提前預約熱門餐廳和環球影城門票
- 下載 Google Maps 和換乘案內 APP
- 準備零錢，許多寺廟和小店只收現金

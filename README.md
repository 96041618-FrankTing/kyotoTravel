# 京都大阪七天六夜旅遊行程APP

一個現代化的旅遊行程應用，使用Vue.js和Tailwind CSS打造，具備地圖導航功能。

## iOS 相容性 ✅

本應用完全支援iPhone和iPad，包括：

### ✅ 支援功能
- **PWA功能**: 可安裝到主畫面，離線使用
- **地圖功能**: Leaflet地圖完全相容，支援觸控操作
- **位置服務**: Geolocation API正常運作（需要允許位置權限）
- **觸控體驗**: 優化的觸控手勢和滾動體驗
- **Service Worker**: 快取和離線功能正常運作

### 🔧 iOS 特殊優化
- 防止Safari自動縮放輸入框
- 支援安全區域（Safe Area）適配
- 優化的觸控滾動體驗
- iOS專屬的PWA meta標籤
- 更好的錯誤處理和權限請求

### 📱 在iPhone上安裝

1. **使用Safari開啟**: `https://your-domain.com`
2. **點擊分享按鈕** (中間的方形箭頭圖標)
3. **選擇"加入主畫面"**
4. **確認安裝**

### ⚠️ 已知限制

- **Service Worker**: iOS 11.3以下版本有限支援
- **位置權限**: 必須使用HTTPS或localhost
- **快取策略**: 針對iOS Safari優化

## 功能特色

- 📱 **手機APP風格介面** - 現代化的響應式設計
- 🗺️ **互動式地圖** - 整合Leaflet地圖，顯示行程地點
- 📍 **自動地點標記** - 自動在當天行程中標記所有地點
- 📍 **用戶位置顯示** - 顯示您現在的位置
- 📅 **橫向滑動日期選單** - 像手機APP一樣的日期導航
- ⏰ **即時倒計時** - 顯示距離出發還有多少時間
- 🌤️ **天氣資訊** - 顯示當前天氣狀況

## 技術架構

- **前端框架**: Vue.js 3
- **樣式框架**: Tailwind CSS
- **地圖服務**: Leaflet
- **建置工具**: Vite
- **PWA支援**: Service Worker

## 安裝與運行

### 環境需求

- Node.js 16+
- npm 或 yarn

### 安裝步驟

1. **安裝依賴**
   ```bash
   npm install
   ```

2. **啟動開發服務器**
   ```bash
   npm run dev
   ```

3. **建置生產版本**
   ```bash
   npm run build
   ```

4. **預覽生產版本**
   ```bash
   npm run preview
   ```

## 行程資料

應用包含完整的7天行程資料：
- **Day 1**: 桃園 → 京都 (清水寺、祇園)
- **Day 2**: 京都景點巡禮 (金閣寺、伏見稻荷大社)
- **Day 3**: 京都文化體驗 (京都御所、二條城、藝伎體驗)
- **Day 4**: 京都 → 大阪 (大阪城、心齋橋)
- **Day 5**: 環球影城
- **Day 6**: 天橋立一日遊
- **Day 7**: 大阪 → 返程

## 開發說明

### 新增行程資料

在 `src/App.vue` 的 `itineraryData` 物件中新增或修改行程資料。每個行程項目應包含：
- `time`: 時間
- `title`: 標題
- `description`: 描述
- `transport`: 交通方式
- `location`: 地點
- `duration`: 預計時間
- `coordinates`: 經緯度座標

### 自定義樣式

使用Tailwind CSS類別進行樣式自定義。顏色主題定義在 `tailwind.config.js` 中。

## 🚀 快速開始

### 線上訪問

直接開啟 `index.html` 檔案或部署到任何網頁伺服器。

### 本地開發

```bash
# 使用 Python 簡易伺服器
python -m http.server 8000

# 或使用 Node.js
npx http-server
```

然後在瀏覽器開啟 `http://localhost:8000`

### 安裝為 PWA

1. 在支援的瀏覽器中開啟網站
2. 點擊「安裝」按鈕或瀏覽器的「安裝應用程式」選項
3. 應用程式將新增到您的主畫面
